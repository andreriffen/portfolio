Olá!

Meu nome é Andre Guilherme Barreto de Farias, escrevo códigos e adoro conversar sobre tecnologia.

Atualmente faço uso de PHP, Java e Banco de Dados MySQL relacional. Recentemente tenho experimentado Javascript em React e Vue. Minhas bibliotecas favoritas são Tailwind, DaisyUI, Sass e Bootstrap.
Minha carreira em desenvolvimento começou em 2022 ingressando no Instituto Federal de Santa Catarina (IFSC). Certamente foi a melhor decisão que tomei! Hoje dedico uma boa parte dos meus dias em programação e código.

Linguagens e tecnologias

Estudos em diferentes linguagens de programação abriram janelas e ampliaram meus horizontes.
As principais abordagens mais utilizadas são:

  • Programação Web com PHP (usando POO e SQL);
  • Programação Orientada à Objetos com Java;
  • Banco de Dados SQL relacional;
  • Análise de Sistemas com UML;
  • Web Design com HTML e CSS;
  • Interação Humano-Computador (Ux);
  • Design Gráfico (Ui);

No momento estou buscando trabalhar pela primeira vez com desenvolvimento de software a nível de estágio, bolsa ou dev júnior.
